Highlighted Project

The Astro-Fuel Project was designed to be and educational site that explore the science and knowledge of outer space. While the main purpose is for learning aspects of it weren’t meant to be engaging in an artful tone. Overtime the designing process and overall experience in creating website and improved, finding better ways and stepping into the industry standard.

The project entail creating 6 pages each that were design in various ways. Using bootstrap as a framework, the project was made in a mobile-first design. With that in mind the design process was first conducted to make sure that phone users were cater to.

Like most site, creating a theme for the site was undoubtably the biggest challenge to overcome. Always as adding a transition to object in order to create a more immersive webpage.

From the project, I was able to acquire more information on how certain animation found in other web pages are taken pages. Research more into java script to help create the desired functionality of the site. Adapting to Sass from CSS for better organization of styling. All in all, this was a better enjoyable project to do.
